## ⚠️ 待优化问题

### P0 - 紧急（影响用户体验）

#### 1. AI服务部署验证
**问题**: 代码已修复，但ECS上是否生效需确认  
**影响**: AI对话可能仍返回默认回复  
**解决方案**:
```bash
# 在ECS上运行测试
bash /opt/lsjy-app/test-backend.sh
```

#### 2. Token安全性
**问题**: 使用简化版Token（`jwt_<userId>_<timestamp>`），非真正JWT  
**风险**: 可被伪造，安全性低  
**解决方案**: 升级到真正的JWT（jsonwebtokens库）  
**工时**: 2小时

### P1 - 重要（影响收入）

#### 3. 支付流程自动化
**问题**: 手动审核截图，无法自动到账  
**影响**: 用户体验差，需要人工介入  
**解决方案**:
- 方案A: 接入微信支付/支付宝API（需要营业执照）
- 方案B: 接入第三方聚合支付（如PayJS、虎皮椒）
- 方案C: 保持现状，优化后台审核流程

**工时**: 
- 方案A: 3-5天（需要申请商户号）
- 方案B: 1天（快速接入）
- 方案C: 0（已完成）

#### 4. 订单管理优化
**问题**: 缺少订单详情、退款、导出功能  
**解决方案**: 完善订单管理后台  
**工时**: 4小时

### P2 - 技术债（影响扩展性）

#### 5. 数据库迁移
**问题**: 使用JSON文件存储，大量数据时性能差  
**影响**: 1000+用户时查询变慢  
**解决方案**: 迁移到SQLite（轻量）或MySQL（生产级）  
**工时**: 1-2天

#### 6. API文档
**问题**: 缺少标准化API文档  
**影响**: 前后端联调效率低  
**解决方案**: 使用Swagger生成API文档  
**工时**: 4小时

#### 7. 日志系统
**问题**: 日志输出到控制台，未持久化  
**影响**: 问题排查困难  
**解决方案**: 接入Winston日志库，输出到文件  
**工时**: 2小时
