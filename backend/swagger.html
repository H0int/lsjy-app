🚀 罗圣纪元 SaaS 平台 API 文档

Base URL: http://localhost:3000/api/v1

🔐 认证授权

POST /auth/login
用户登录 - 返回 accessToken 和 refreshToken
POST /auth/register
用户注册
POST /auth/refresh
刷新 Token
POST /auth/logout 需要认证
退出登录

👤 用户管理

GET /users/me 需要认证
获取当前用户信息
PUT /users/me 需要认证
更新个人信息
GET /users 管理员
获取用户列表（管理员）

🤖 AI 工具

GET /ai/categories
获取工具分类列表
GET /ai/tools
获取 AI 工具列表
GET /ai/tools/:id
获取工具详情
POST /ai/tools/:id/call 需要认证
调用 AI 工具
GET /ai/history 需要认证
获取调用历史
GET /ai/models
获取可用模型列表
GET /ai/providers
获取 AI Provider 状态

💰 支付中心（圣点）

GET /payment/coin/balance 需要认证
查询圣点余额
GET /payment/coin/packages
获取充值套餐列表
POST /payment/coin/recharge 需要认证
创建充值订单
GET /payment/coin/transactions 需要认证
查询圣点交易记录
GET /payment/orders 需要认证
查询支付订单

🛠️ 管理后台

GET /admin/dashboard/stats 管理员
获取仪表盘统计数据
GET /admin/announcements 管理员
获取公告列表
POST /admin/announcements 管理员
创建公告
GET /admin/coupons 管理员
获取优惠券列表
GET /admin/campaigns 管理员
获取活动列表
GET /admin/tickets 管理员
获取工单列表
GET /admin/faqs 管理员
获取 FAQ 列表
GET /admin/automation-rules 管理员
获取自动化规则列表
GET /admin/moderation 管理员
获取内容审核列表

⚙️ 系统管理

GET /system/configs/public
获取公开配置
GET /system/configs 超级管理员
获取所有配置
PUT /system/configs/:key 超级管理员
更新配置
GET /system/logs 管理员
获取操作日志
GET /notifications 需要认证
获取通知列表

🔑 角色权限

GET /roles 管理员
获取所有角色
GET /roles/permissions 管理员
获取所有权限
POST /roles 管理员
创建角色
POST /roles/assign 管理员
分配角色给用户

🏥 健康检查

GET /health
服务健康检查
