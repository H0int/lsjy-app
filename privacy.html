罗圣纪元 隐私政策

生效日期：2026年6月3日 | 最近更新：2026年6月3日

罗圣纪元互联网科技有限责任公司（以下简称"我们"）深知个人信息对您的重要性，并会尽全力保护您的个人信息安全。本隐私政策适用于您通过罗圣纪元移动应用（以下简称"本应用"）使用我们的产品和服务。

一、我们收集的信息

本应用采用本地存储技术，大部分数据保存在您的设备本地，不会上传至我们的服务器。我们可能收集以下信息：

  • 账户信息：您注册时提供的用户名、密码（经加密处理后的哈希值）、手机号。
  • 使用数据：您在本应用中创建的内容、工具使用记录、算力余额与兑换记录，均存储在您的设备本地。
  • 设备信息：设备型号、操作系统版本、屏幕分辨率（仅用于界面适配）。
  • 第三方服务数据：当您使用 AI 智能功能时，您的输入内容会发送至火山引擎 API 进行处理，具体隐私政策请参阅 火山引擎隐私政策。

二、信息的存储

您的个人信息和使用数据主要通过浏览器/WebView 的 localStorage 技术存储在您的设备本地。我们不会主动将您的数据上传至外部服务器，但以下情况除外：

  • AI 功能调用时，输入文本会通过加密通道（HTTPS）发送至火山引擎 API，且不保存对话记录。
  • 短视频解析功能可能请求第三方解析接口获取视频信息。

三、信息的使用

我们收集的信息仅用于以下目的：

  • 提供、维护和改进本应用的核心功能服务
  • 处理您的 AI 内容生成、视频解析等工具请求
  • 账户安全管理与算力/卡密系统运营
  • 处理您的反馈和客户服务请求

四、信息的共享

我们不会将您的个人信息出售或出租给第三方。仅在以下必要场景中共享：

  • AI 服务提供商：火山引擎（字节跳动），用于处理 AI 内容生成请求
  • 法律要求：在法律法规要求或政府主管部门依法要求时

五、您的权利

您对自己的个人信息享有以下权利：

  • 访问权：您可以在本应用的"个人中心"查看您的账户信息和使用数据
  • 修改权：您可以在应用内修改个人信息
  • 删除权：您可以清除应用本地数据（设置 > 清除数据），或联系我们请求删除
  • 注销权：您可以联系客服注销您的账户，我们将在核实后删除相关数据

六、安全措施

我们采取以下安全措施保护您的信息：

  • 密码采用加密存储，不以明文保存
  • 所有网络请求使用 HTTPS 加密传输
  • API Key 等敏感凭证不暴露在前端代码中（服务端代理）
  • 定期更新应用以修复安全漏洞

七、未成年人保护

我们高度重视未成年人个人信息保护。若您是未满 14 周岁的未成年人，请在监护人的指导下使用本应用，并在取得监护人同意后提供个人信息。

八、隐私政策的更新

我们可能会不定期更新本隐私政策。更新后的政策将在本应用内公示，重大变更将通过弹窗通知。建议您定期查阅本政策以了解最新信息。
